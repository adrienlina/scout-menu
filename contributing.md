## Appel à contributeurs — Scout Menu

Scout Menu est une application web open source pensée pour les intendant·e·s en camp : planification des menus, listes de courses, gestion des stocks, calcul de l'empreinte carbone des repas, etc.

Le projet est en développement actif et cherche des contributeur·ice·s, qu'ils·elles soient techniques ou fonctionnel·le·s.

Côté fonctionnel : nous cherchons des personnes avec une expérience terrain : chef·fe·s de troupe, intendant·e·s de camps, cuisinier·ère·s bénévoles. Vous apporterez au projet cas d'usage, retours
utilisateur·ice·s, idées de fonctionnalités, tests en conditions réelles.

Côté technique, l'objectif est d'utiliser des outils de génération de code par IA pour environ 90 % du travail d'implémentation - afin d'économiser notre temps. La stack est moderne (React, TypeScript,
Tailwind, Supabase), et parfois il faudra se plonger dedans. Ce qui compte, c'est la capacité à formuler clairement les besoins, relire du code généré, valider que le résultat correspond à l'intention, et la
communication avec les autres membres du projet.

En pratique, contribuer peut vouloir dire : ouvrir une issue bien rédigée, proposer une maquette, tester une fonctionnalité, ou piloter un agent IA pour implémenter une feature en moins d'une heure.
